# 🚀 Desafio: 30 Dias, 30 Sites

Bem-vindo ao repositório do meu desafio pessoal de desenvolvimento web! O objetivo aqui é construir **30 sites/projetos diferentes em 30 dias**, focando em design, responsividade e performance.

Este repositório serve como um portfólio vivo, demonstrando minha evolução técnica e criatividade ao longo do processo.

---

## 🛠️ Tecnologias Utilizadas

Para garantir máxima performance e aprendizado da base, a maioria dos projetos utiliza:
- **HTML5** Semântico
- **CSS3** (Variáveis, Flexbox, Grid e animações modernas)
- **JavaScript** Vanilla (Puro)
- **Hospedagem:** Cloudflare Pages (CDN Global)

Este desafio não é apenas sobre quantidade, mas sobre **consistência**. A cada dia, busco resolver um problema de design diferente ou implementar uma funcionalidade nova em JavaScript, aprimorando meu workflow de desenvolvimento.
